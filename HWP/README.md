# HWP Studio – 한국어 문서 편집기

Python 기반 Hangul(HWP) 호환 문서 편집기입니다.
Windows에서 네이티브로 실행됩니다.

---

## 🚀 실행 방법

### 방법 1 (권장): 배치 파일
`run.bat` 를 더블클릭하세요.
자동으로 필요한 패키지를 설치하고 실행합니다.

### 방법 2: 직접 실행
```
pip install python-docx reportlab pillow
python main.py
```

---

## 📋 요구 사항

- Python 3.8 이상
- Windows 10/11 (tkinter 기본 포함)
- 인터넷 연결 (최초 패키지 설치 시)

---

## ✨ 주요 기능

| 기능 | 설명 |
|------|------|
| **다중 탭** | 여러 문서를 탭으로 동시 편집 |
| **체크박스** | ☐/☑ 클릭으로 토글 |
| **표 삽입** | 행×열 지정하여 ASCII 표 삽입 |
| **서식** | 굵게, 기울임, 밑줄, 취소선 |
| **글꼴** | 맑은 고딕, 굴림 등 한국어 폰트 지원 |
| **정렬** | 왼쪽/가운데/오른쪽/양쪽 정렬 |
| **색상** | 글자색, 형광펜 색 |
| **찾기/바꾸기** | 전체 치환 지원 |
| **내보내기** | PDF, DOCX 내보내기 |
| **저장 형식** | .hwps (JSON 기반 자체 형식) |
| **실행 취소** | Ctrl+Z / Ctrl+Y |
| **확대/축소** | 75% ~ 200% |

---

## ⌨️ 단축키

| 단축키 | 기능 |
|--------|------|
| Ctrl+N | 새 문서 |
| Ctrl+O | 파일 열기 |
| Ctrl+S | 저장 |
| Ctrl+B | 굵게 |
| Ctrl+I | 기울임 |
| Ctrl+U | 밑줄 |
| Ctrl+H | 찾기/바꾸기 |
| Ctrl+Z | 실행 취소 |
| Ctrl+Y | 다시 실행 |
| Ctrl++ | 확대 |
| Ctrl+- | 축소 |
| F1 | 도움말 |

---

## 📁 파일 형식

- **.hwps** – HWP Studio 기본 형식 (JSON, UTF-8)
- **.txt / .md** – 텍스트 파일 열기
- **.docx** – Word 문서 열기/내보내기
- **.pdf** – PDF 내보내기

---

## 💡 체크박스 사용법

1. `삽입` 메뉴 → `체크박스 삽입` 또는 도구모음의 `☑ 체크박스` 버튼
2. 문서에 `☐` 가 삽입됩니다
3. 해당 문자를 **클릭**하면 `☑` 로 토글됩니다
4. 다시 클릭하면 `☐` 로 돌아옵니다
